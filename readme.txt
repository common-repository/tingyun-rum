== Description ==
This Plugin will add Tingyun Browser RUM (Real User Monitoring) Agent on your WordPress website.

== Installation ==
1. If your do not have yet, sign in for a new Tingyun account (https://account.tingyun.com/reg/register?userFrom=wp_plugin#/) and create a new Tingyun Browser Application.
2. Install this WordPress plugin
	- After login WordPress admin: 'Plugins' > 'Add new'. Search for "Tingyun Browser" and "Install Now", or
	- Upload entire directory to <your-wordpress-root-path>/wp-content/plugins
3. After login WordPress admin, goto 'Plugins' > 'Installed Plugins' page, activate "Tingyun Browser Plugin for WordPress" and configure through the "Tingyun Browser" sub-menu under "Settings. You'll need your Application's ID and Key create in Step 1.
